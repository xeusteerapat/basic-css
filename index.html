CSS syntax ประกอบด้วยอะไรบ้าง CSS syntax ประกอบด้วย selector และ declaration โดยที่ selector คือ การเลือก element ที่ต้องการปรับแต่ง ส่วน declaration คือการประกาศลักษณะของ element นั้น ๆ โดย มี 2 องค์ประกอบหลักคือ
  • property: คุณสมบัติ (background, color)
  • value: ค่าที่กำหนด (#fff, 20px)
    •             
                table { 
                    border-collapse: 
                    collapse; width: 100%; 
                }
    การเขียน CSS มีกี่แบบ อะไรบ้าง
    1. inline CSS: เขียนใน style tag ในไฟล์ HTML
    2. external CSS: นำเข้าไฟล์ CSS จากภายนอก
    3. internal CSS: เขียนภายใน HTML element
    selector มีกี่แบบ ประกอบด้วยอะไรบ้าง
    1. ID : การกำหนดชื่อของ element โดยจะมีได้แค่ element เดียวเท่านั้น
    2. Class : การกำหนดชื่อของ element โดยสามารถมีได้หลาย element
    3. Pseudo element : การกำหนดเฉพาะบางส่วนของ element
    4. Pseudo class : การกำหนดเฉพาะบางสถานะของ class
    5. Combinators : การกำหนดเงื่อนระหว่าง 2 selector
    6. attribute selector : การกำหนดการเลือกชื่อ attribute ใน element
    combinator มีกี่แบบ ประกอบด้วยอะไรบ้าง
    1. descendant selector (space) : a b = tag b ทั้งหมดที่อยู่ใน a
    2. child selector (>) : a > b = tag b ที่เป็นลูกของ a ie. 
      
                .container > p { 
                  max-width: 1100px; 
                }
    3. adjacent sibling selector (+) : a + b = tag b ที่อยู่ล่าง a และติดกับ a เท่านั้น
    4. general sibling selector (~) : a ~ b = tag b ที่อยู่ล่าง a ทุก tag
